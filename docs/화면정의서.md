##  화면 초안
### 목표
- Figma에서 정형화된 형식으로 화면 설계서를 작성한다.
- Frame별로 기능 구현 정의 템플릿을 세팅하여 개발 진행 상황을 추적한다.

### 기술 스택
- React 19.2
- Backend WAS 서버 별도 존재
  - Backend Server 정보는 설정 값으로 할수 있도록 구성
  - API 결과값은 Json 형식으로 받음.

### Backend 서버 정보 
IP:192.168.50.114
PORT:8080

### 문서 구조
```
#### {번호}. {화면명}
| 상태 | Frame Name | Node ID |

##### 레이아웃 구성 (선택)
| 영역 | 컴포넌트 | 설명 |

##### 기능 정의
| 기능 | 설명 | 구현 상태 |
```
- 구현 상태: `[ ]` 미완료, `[x]` 완료

##### 레이아웃 구성
| 영역 | 컴포넌트 | 설명 |
|------|----------|------|
| 헤더 | Header | 로고, 네비게이션, 사용자 정보 |
| 상단 좌측 | 날씨 카드 | 현재/내일 날씨 (온도, 습도, 풍속) |
| 상단 우측 | 실시간 차압 및 센서 정보 | 연돌 발생 현황, 디바이스 정보 |
| 중단 | 실내 차압 카드 | 히트맵, 연돌 압력 프로파일, 상세 데이터 |
| 하단 상 | 연돌 발생 이력 | 테이블 형식 이력 조회 |
| 하단 하 | 통계 그래프 | 평균 기압/기온/습도 그래프 |

| 기능 | 설명 | 구현 상태 |

| 상세 데이터 테이블 | 층별 온도 데이터 (실내/통로/온도차) | [ ] |
| 연돌 발생 이력 | Device ID, Zone, 상태, 압력, 발생 일시 | [ ] |
